简介

棋眼 Pro 是一个融合计算机视觉、STM32 机械臂控制与博弈 AI 的智能五子棋对弈系统。
系统通过摄像头自动识别真实棋盘与棋子位置，上位机完成棋局建模和落子决策，并通过串口控制机械臂在实体棋盘上自动落子。项目支持人机对弈、机机自博弈和对局记录/回放，为智能棋类机器人、工业视觉与嵌入式控制的综合实践提供一个完整案例。

英文简介

Qiyan Pro is an intelligent Gomoku system that integrates computer vision, a STM32-based robot arm and game-search AI.
A camera detects the real board and stones, the host program models the game state and computes moves, then sends commands to the robot arm to place stones on the physical board. The project supports human-vs-AI and AI-vs-AI modes with game recording and replay, serving as a complete hands-on project for vision-guided robotics and embedded control.
